# Al Maha Foods - React Application

Professional website for Al Maha Foods - a leading Basmati Rice exporter and QA services provider from India.

## Project Information

- **Converted from:** WordPress HTML
- **Framework:** React 18 with Vite
- **Routing:** React Router v6
- **Styling:** CSS Modules
- **Maps:** amCharts 4
- **Slider:** Swiper

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

Builds the app for production to the `dist` folder.

### Testing

```bash
npm run test:visual
```

Runs Playwright visual regression tests.

## Project Structure

```
src/
├── pages/          # Page components
├── components/     # Reusable components
├── styles/         # CSS modules and global styles
├── App.jsx         # Main app component with routes
└── main.jsx        # Entry point

public/
└── assets/         # Static assets (images, fonts, etc.)
```
